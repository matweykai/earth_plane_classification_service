# Сервис выставления тегов для спутниковых снимков

Данный сервис позволяет загружать картинку и получать набор тегов, которые были обнаружены нейросетью

## Структура репозитория

```
.
├── config
├── src
│   ├── containers
│   ├── routes
│   └── services
└── weights
```
